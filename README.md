# Glueball_collab

Small team repo for glueball-related calculations and reading.

## What's where

- `docs/shared/`: official notes
- `docs/notes/`: our own notes, meeting minutes, ideas, etc.
- `docs/literature/`: external papers.
- `src/external/`: code from elsewhere.
- `src/glueball/`: our own code.
- `results/`: figures, tables, output data, etc., if applicable.
